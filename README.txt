=== Bukkit Auto-installer ===

= SAYQWACA ? =

Bukkit Auto-installer est un script python dont la fonction principale est de telecharger la derniere version du serveur bukkit (pour minecraft), de l'installer, de le configurer puis de telecharger la derniere version de plugins essentiels ! Ce script peut aussi generer des statistiques (Grace a MCLP (minecraft log parser - https://github.com/stevenleeg/Minecraft-Log-Parser) interresantes pour l'administration du serveur ! ) 


= REQUIS =

- Systeme basé sur unix (desolé utilisateurs de windows, ce script n'est pas pour vous) de preferance mac os X 10.6 (testé)
- Savoir lancer du python
- 10 minutes


= INSTALATION =

Ouvrez un terminal
Tapez chmod +x (notez l'espace après le x) puis glissez y le fichier installer.command
Tapez entrer
Ouvrez installer.command
Enjoy !
